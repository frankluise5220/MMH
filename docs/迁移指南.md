
## 迁移步骤

### 在新机器上

```bash
# 1. 拷贝上述文件到新目录

# 2. 安装依赖
npm install

# 3. 配置 .env 指向新数据库（或启动 Docker）
docker-compose up -d

# 4. 推送数据库结构
npx prisma db push

# 5. 构建
npm run build

# 6. 启动
npx next dev -H 0.0.0.0 -p 7777
# 或生产模式: npx next start -H 0.0.0.0 -p 7777
```

### 数据库迁移

项目使用 `prisma db push`（无 migration 文件）。方式二选一：

**方式 A：同步空库结构（新建环境）**
```bash
npx prisma db push
```
数据库中无历史数据，从头开始。

**方式 B：迁移全部数据**
```bash
# 在原机器导出
pg_dump -h <旧主机> -U <用户> -d <库名> --no-owner --no-acl > wiseme_dump.sql

# 在新机器导入
psql -h <新主机> -U <用户> -d <库名> < wiseme_dump.sql

# 验证结构一致
npx prisma db push  # 应该显示 "Nothing to change"
```
